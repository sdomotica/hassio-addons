# 🏠 Sdomotica Home Assistant Add-ons

Benvenuti nel repository ufficiale degli add-on **Sdomotica**. Questa raccolta è pensata per estendere le potenzialità di Home Assistant (Hassio/HassOS), integrando servizi e applicazioni specifiche per il monitoraggio e il controllo domotico avanzato.

---

## 🚀 Overview degli Add-ons
Qui trovi una selezione di strumenti pronti all'uso per il tuo ecosistema smart:

| Add-on | Descrizione | Stato |
| :--- | :--- | :--- |
| **Sdomotica App** | Il core delle logiche di integrazione personalizzate. | ✅ Stabile |
| **Web Server** | Interfaccia web dedicata per la gestione dei servizi Sdomotica. | 🌐 Online |
| **Websocket Service** | Servizio di comunicazione real-time per aggiornamenti istantanei. | ⚡ Active |
| **Hardware Monitor** | Tool avanzato per il monitoraggio delle risorse del tuo Raspberry/NUC. | 📊 Monitor |

---

## 🛠 Installazione Rapida
Aggiungere questi add-on al tuo Home Assistant è semplicissimo:

1. Vai nella dashboard di **Home Assistant**.
2. Naviga in **Impostazioni** > **Componenti aggiuntivi**.
3. Clicca su **Store dei componenti aggiuntivi** (in basso a destra).
4. Clicca sul menù a tre puntini (⋮) in alto a destra e seleziona **Repository**.
5. Incolla il seguente link:
   ```text
   [https://github.com/sdomotica/hassio-addons](https://github.com/sdomotica/hassio-addons)
