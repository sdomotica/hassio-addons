## [EN] Inim PrimeX

The PrimeX represents the next generation of Inim's security and home-automation platforms. It is more than just a control panel; it is a fully integrated hub designed to provide a comprehensive lifestyle experience. PrimeX is built on the solid foundation of Inim's expertise, enhanced by a powerful new processor and native connectivity options that set a new standard for speed, reliability, and flexibility.

Safety is guaranteed at the highest levels, with the system complying with Grade 3 of the EN50131 standard and Grade ATS6 of EN50136, ensuring the most reliable alarm communications available on the market.

## [EN] Connectivity & Features

Unlike previous models that required the PrimeLAN module for advanced IP functions, the PrimeX series integrates advanced connectivity directly on the main board.

PrimeX features native Ethernet connectivity and an onboard Web Server with the intuitive "Alien" interface. It supports interactive graphic maps, advanced video verification with ONVIF webcams (JPEG/MJPEG), and native integration with KNX systems.

## [IT] Connettività e Funzionalità

A differenza dei modelli precedenti che richiedevano il modulo PrimeLAN per le funzioni IP avanzate, la serie PrimeX integra la connettività avanzata direttamente sulla scheda principale.

PrimeX dispone di connettività Ethernet nativa e di un Web Server integrato con l'interfaccia intuitiva "Alien". Supporta mappe grafiche interattive, video-verifica avanzata con telecamere ONVIF (JPEG/MJPEG) e integrazione nativa con i sistemi KNX.
